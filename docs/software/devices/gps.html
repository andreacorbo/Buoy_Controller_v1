GPS - Global Position System¶

Standard GPS device:¶

This module contains standard GPS devices tools.

class dev_gps.GPS(instance, tasks=[])[source]¶

Creates a GPS device object.

Extends the device and the tools.nmea classes.

Parameters:

instance str The object instance number (if multiple devices are present) needs a correspondent section in the config_file.

tasks list (optional) A list of methods to be executed at object creation.

start_up()[source]¶

Performs the device specific initialization sequence.

Return:
True or False depends on startup sequence successfull completion.
main(sentence='RMC')[source]¶

Retreives data either from a UART or I2C gps device.

Passes data char by char to tools.nmea.NMEA.get_sentence() to get a valid NMEA string.

Parameters:
sentence str The desired NMEA sentence.
Return:
True or False depends on gps got a valid fix.
